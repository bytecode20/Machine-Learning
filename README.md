# Machine-Learning
💼 Salary Prediction in India using Regression Models
📌 Project Overview

This project focuses on predicting employee salaries in India based on their years of experience and other relevant features. The goal is to explore and compare different regression algorithms to identify which model provides the most accurate salary predictions.

We use a structured dataset containing Salary vs Experience (India-specific context) and apply multiple machine learning techniques to analyze performance and accuracy.

🎯 Objectives
Understand the relationship between experience and salary
Implement multiple regression models
Compare model performance using accuracy metrics
Identify the best-performing algorithm for salary prediction
📊 Dataset

The dataset includes:

Years of Experience
Salary (in INR)
(Optional additional features if included: education, location, role, etc.)
🤖 Models Implemented

The following regression algorithms are trained and evaluated:

Simple Linear Regression
Establishes a direct linear relationship between experience and salary
Multiple Linear Regression
Uses multiple features to improve prediction accuracy
Polynomial Regression
Captures non-linear relationships between variables
Support Vector Regression (SVR)
Uses kernel functions to model complex patterns
Decision Tree Regression
Splits data into decision rules for prediction
Random Forest Regression
Ensemble method using multiple decision trees for improved accuracy
📈 Model Evaluation

Each model is evaluated using:

R² Score
Mean Absolute Error (MAE)
Mean Squared Error (MSE)

The performance comparison helps determine the most suitable model for real-world salary prediction.

This project demonstrates how different regression techniques perform on a real-world inspired dataset. It provides insights into selecting appropriate models based on data characteristics and prediction requirements.
